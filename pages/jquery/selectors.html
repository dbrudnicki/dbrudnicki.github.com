jQuery - Selectors


Types of Selectors

There are basically four types of selectors in jQuery:
  • By ID - This type of selector is preceded by a pound symbol: #
  • By class - This type of selector is preceded by a decimal point: .
  • By attribute - This type of selector is wrapped in brackets: [...]
  • jQuery custom - This type of selector is preceded by a colon: :
Given the following HTML, I'll show examples of each selector:
	<html>
		<body>
			<input id="testButton" type="button" val="Click Me"/>
			<br/>
			<div>
				<span class="odd">First Line</span>
				<span class="even">Second Line</span>
				<span class="odd">Third Line</span>
			</div>
			<br/>
			<label>Yes <input="yesRadio" id type="radio" name="yesno" checked="checked"/></label>
			<label>No <input id="noRadio" type="radio" name="yesno"/></label>
		</body>
	</html>

Examples:

	//To apply a click event to the button with the id of "testButton":
	$("#testButton).click();
	
	//To highlight the rows marked with the class="odd" in the div:
	$(".odd").css("background-color", "#ADD8E6");
	
	//To select all inputs that have an attribute of type equal to "radio":
	$("input [type='radio']").each();
	
	//To get the checked radio:
	$(":checked").each();

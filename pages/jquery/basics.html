jQuery Basics


Step 1:

The best place to start is at the beginning. First your HTML page will need a reference either to the online
copy of jQuery or to a downloaded version. Here is an example of a reference to the online version:

	<script type="text/javascript" src="http://code.jquery.com/jquery-1.7.1.min.js"></script>

Step 2:

Once you've included the jQuery script into your page you can access the framework by calling the
jQuery variable also known as the $ variable.
($ is just an alias for jQuery which I discuss in the No Conflict tutorial).

The recommended place to start using jQuery is on the document ready. When the document is ready the
DOM (Document Object Model), will have been loaded even though the graphics may still be loading. The
code to plug in to the document ready is:

	<script type="text/javascript">
		$(document).ready(function() {
			//do stuff here
		});
	</script>

So, what does this all mean? The $() is a selector. In this case we are selecting the DOM document itself.
Next, we are saying, once the document is ready execute any commands in supplied the anonymous function.

For Example, click the button:
Click Me!

The code to do this is:

	<script type="text/javascript">
		$(document).ready(function() {
			$("#myButton").click(function() {
				alert("Hello World!");
			});
		});
	</script>
	...
	<body>
		<input id="myButton" type="button" value="Click Me!"/>
	</body>

So, once the document is ready we add a "click" event handler to the selected button using the CSS
# selector. (More on that in the next tutorial on Selectors.)
